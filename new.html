Area Coverage Demo

Project Overview:

In this project, I designed and developed an autonomous robot intended to be equipped with a pressure washer system for cleaning pool decks. The main challenge was to create a system that could navigate around obstacles while ensuring complete area coverage.

Key Contributions:

  • Area Coverage Algorithm: I developed an area coverage algorithm using the A* Search method that was integrated into Madara Middleware. This technique to allow the robot to navigate efficiently, avoid obstacles, and fully cover a defined area.
  • LiDAR Integration for Object Detection: I utilized LiDAR to capture point clouds and extract clusters of objects. This was used to determine where objects are located on a 2d grid map.
  • GPS Waypoint Generation: This method also incorporated a gps system using Emlid Reach RS3 to determine precise latitude and longitude of objects. The recievers identified the location of the robot and this allowed us to convert the LiDAR cartesian coordinates to GPS coordinates.

Tools & Technologies:

  • Livox Mid-360 (LiDAR): LiDAR sensor for high-precision 3D mapping.
  • Emlid Reach RS3: GNSS receiver for accurate positioning.
  • Reach M2 LoRa radios: Long-range communication system for data transmission.
  • NVIDIA Jetson Orin: AI computing platform for real-time processing and analysis.
  • C++
  • CTest
  • Madara Middleware (madara.ai): Middleware for multi-agent systems and autonomous applications.

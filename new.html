In this project, I developed an Area Coverage algorithm using A* to avoid objects and cover a defined area. I detected objects using LiDAR to convert point clouds and get clusters of objects. This was used to determine where objects are located on a 2d map. This system also incorporated a gps system to determine latitude and longitude of objects to generate waypoints for the path of the robot.

Tools & Technologies:

  • Livox Mid-360 (LiDAR): LiDAR sensor for high-precision 3D mapping.
  • Emlid Reach RS3: GNSS receiver for accurate positioning.
  • Reach M2 LoRa radios: Long-range communication system for data transmission.
  • NVIDIA Jetson Orin: AI computing platform for real-time processing and analysis.
  • C++
  • CTest
  • Madara Middleware (madara.ai): Middleware for multi-agent systems and autonomous applications.
